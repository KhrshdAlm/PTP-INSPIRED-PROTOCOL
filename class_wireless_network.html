Detailed Description

This class creates Multi-hop Wireless Network.

Nodes are added to the network. Modified PTP protocol runs through this class. Control Packets SYNC, FOLLOW, DREQ, DRPLY are sent using sendSyncFollowPacket, sendDreqPacket and sendDrplyPacket function. Upon receiving packet, receivePacket function is invoked, which extracts info from packet and copies timestamps to Time Vector of recieving node. Object of this class updates local time of each node.

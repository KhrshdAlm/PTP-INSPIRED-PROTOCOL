Detailed Description

This class creates Wireless Homogenous Node in the Network.

Node id, master id, hop of node, Ipv4 address are specified through constructor. Clock Deviation is chosen randomly. It Stores index of Sockets to its neighbour. A vector of Time for Storing timestamps collected from previous hops. It calculates Clock Offset of the Node with respect to master.
